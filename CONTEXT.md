# Project Context for AI Assistants

## Project Overview
이 프로젝트는 멘토-멘티 매칭 플랫폼입니다.

## Architecture
- **Backend**: Node.js/Express API
- **Frontend**: React/Next.js
- **Database**: PostgreSQL
- **Authentication**: JWT

## Key Concepts
- **Mentor**: 경험을 공유하는 사용자
- **Mentee**: 멘토링을 받는 사용자
- **Matching**: 멘토와 멘티를 연결하는 프로세스
